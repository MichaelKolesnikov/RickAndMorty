# Rick and Morty Characters App

Android приложение, отображающее список персонажей из Rick and Morty API. 

## Tech Stack
- **Jetpack Compose**
- **Koin** — DI
- **Room** — local database
- **Ktor Client**
- **Kotlinx.serialization**
- **Coil** — для загрузки изображений
- **ViewModel + StateFlow** 

## Clean Architecture & MVVM
### Dependencies
- app -> presentation
- app -> data
- presentation -> domain
- data -> domain
- data -> common
